# Поликлиника
# Лабораторная работа №4: Kafka и взаимодействие сервисов

## Цель
«Инфраструктура» - Реализация сервиса генерации данных и его интеграция с сервером

## Описание изменений
Добавлено приложение Polyclinic.Infrastructure.PosgreSQL, в котором описываются конфигурации сущностей и датасид для миграций в базу данных.
Вся инфраструктура Inmemory заменена на инфраструктуру, использующую базу данных в качестве места хранения записей о приемах, личной информацией врачей и пациентов.


## Описание изменений
- настроен Polyclinic.AppHost для оркестрации сервисов с Aspire
- настроен запуск контейнеров PostgreSQL и Kafka
- API и Kafka интегрированы в Aspire как управляемые проекты
- через Aspire налажен единый запуск всех компонентов 

### Запуск приложения 

`dotnet run --project Polyclinic.AppHost`

После запуска автоматически поднимаются:
- PostgreSQL
- Kafka
- API Host
- остальные сервисы и слои
