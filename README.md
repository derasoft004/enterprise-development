# Поликлиника
# Лабораторная работа №3: Реализация объектно-реляционной модели

## Цель
«ORM» - Реализация объектно-реляционной модели. Подключение к базе данных и настройка оркестрации

## Описание изменений
Добавлено приложение Polyclinic.Infrastructure.PosgreSQL, в котором описываются конфигурации сущностей и датасид для миграций в базу данных.
Вся инфраструктура Inmemory заменена на инфраструктуру, использующую базу данных в качестве места хранения записей о приемах, личной информацией врачей и пациентов.


## Запуск миграций

** dotnet ef migrations add InitialCreate --startup-project ../Polyclinic.API.Host **


Все миграции сохраняются в Polyclinic.Infrastructure.PosgreSQL/Migrations, после создания миграций с командой 

** dotnet run **

создается БД, заполненная данными из dataseed
