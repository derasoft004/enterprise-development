# Поликлиника

## Цель
Реализация проекта сервисно-ориентированного приложения для управления медицинским учреждением.

## Описание проекта
Система управления поликлиникой включает в себя учет пациентов, врачей и их специализаций, медицинских записей на приемы. В DataSeed хранятся тестовые данные о врачах и их специализациях, пациентах и приемах

## Доменная модель

### Основные сущности:

**Doctor** - характеризует врача поликлиники
- Идентификатор, паспортные данные, ФИО
- Год рождения, специализация, стаж работы

**Patient** - информация о пациенте
- Идентификатор, паспортные данные, ФИО
- Пол, дата рождения, адрес
- Группа крови, резус-фактор, телефон

**Appointment** - информация о медицинском приеме
- Содержит ссылки на пациента и врача
- Дата и время приема, номер кабинета
- Флаг повторного приема

**Specialization** - справочник медицинских специализаций


## Тесты

### PolyclinicTests - юнит-тесты

**DoctorsExperienceMore10** - Вывести врачей со стажем работы более 10 лет.

**PatientsByDoctor** - Получить список пациентов конкретного врача.

**RepeatAppointmentsCountLastMonth** - Подсчитать количество повторных приемов за последний месяц.

**PatientsAgeMore30WithMultipleDoctors** - Найти пациентов старше 30 лет, посещавших нескольких врачей.

**AppointmentsInRoomForChosenMonth** - Получить информацию о приемах в конкретном кабинете за выбранный месяц.

## LINQ-запросы

### Queries - LINQ-запросы для анализа данных

**DoctorsWith10Plus** - Врачи с опытом работы 10+ лет

**PatientNamesForDoctor** - Имена пациентов по конкретному врачу

**FollowUpsLastMonth** - Количество повторных приемов за последний месяц

**Patients30PlusMultiDoctors** - Пациенты старше 30 лет, посещавшие нескольких врачей

**ThisMonthInRoom** - Приемы в конкретном кабинете за текущий месяц
